## Platform Details

### Amazon Bedrock

- **Bedrock Mantle launched** (Dec 3, 2025): Full OpenAI-compatible inference engine with Responses API, Chat Completions API, asynchronous inference, stateful conversation management, and simplified tool use integration. Three new IAM policies: AmazonBedrockMantleFullAccess, AmazonBedrockMantleReadOnly, AmazonBedrockMantleInferenceAccess. Source: https://docs.aws.amazon.com/bedrock/latest/userguide/bedrock-mantle.html
- **Converse API batch inference** (Feb 27, 2026): Converse API format can now be used for batch inference input data, providing consistent request format across models. Source: https://docs.aws.amazon.com/bedrock/latest/userguide/batch-inference-data.html
- **Fine-tuning via OpenAI-compatible APIs** (Feb 17, 2026): Reinforcement fine-tuning for open-weight models through Files API, fine-tuning jobs APIs, and inference APIs. Source: https://docs.aws.amazon.com/bedrock/latest/userguide/fine-tuning-openai-apis.html
- **Reinforcement fine-tuning** (Dec 3, 2025): New capability to improve foundation model performance through feedback signals. Source: https://docs.aws.amazon.com/bedrock/latest/userguide/reinforcement-fine-tuning.html
- **Guardrails code security enhancements** (Nov 19, 2025): Standard tier enhanced with code content filters and prompt leakage detection for coding use cases. Source: https://docs.aws.amazon.com/bedrock/latest/userguide/guardrails-code-domain.html
- **Cross-account Guardrails sharing** (Nov 21, 2025): Guardrails can now be shared across accounts within an AWS organization. Source: https://docs.aws.amazon.com/bedrock/latest/userguide/guardrails-resource-based-policies.html
- **Data Automation enhancements**: Blueprint instruction optimization with ground truth labeling (Dec 18, 2025). Synchronous invocation support (Nov 20, 2025). Source: https://docs.aws.amazon.com/bedrock/latest/userguide/doc-history.html
- **Priority, Flex, and Reserved service tiers** (Nov 18-26, 2025): Three new inference service tiers for on-demand and reserved capacity. Source: https://docs.aws.amazon.com/bedrock/latest/userguide/doc-history.html
- **Region expansion** (Nov 18, 2025): Africa (Cape Town), Asia Pacific (New Zealand), Canada West (Calgary), Mexico (Central), Middle East (Bahrain). Source: https://docs.aws.amazon.com/bedrock/latest/userguide/doc-history.html

### Amazon Bedrock AgentCore

- **Evaluate API action** added to Data Plane operations — new API for agent evaluation not previously in the API reference. Source: https://docs.aws.amazon.com/bedrock-agentcore/latest/APIReference/API_Evaluate.html

### Amazon Q

- **Account subscription limit doubled** (Feb 18, 2026): Maximum increased from 10 to 20 accounts per AWS Region per organization. Source: https://docs.aws.amazon.com/amazonq/latest/qdeveloper-ug/doc-history.html
- **GitHub conversation-based feedback** (Dec 29, 2025): Feature development workflow now uses /q command for conversation-based feedback instead of traditional GitHub review process. Source: https://docs.aws.amazon.com/amazonq/latest/qdeveloper-ug/github-feature-development.html

### Google Vertex AI

- **Agent Designer** (Dec 19, 2025): New low-code visual tool for building agents, available in preview. Source: https://docs.cloud.google.com/agent-builder/agent-designer
- **Cloud API Registry** (Dec 18, 2025): New API management service available in preview. Source: https://docs.cloud.google.com/api-registry/docs/overview
