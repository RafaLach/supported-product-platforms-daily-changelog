- **[ENDPOINT] [MEDIUM] OpenAI Codex MCP Apps Resource Reads and Tool Metadata**: Codex v0.120.0 expands MCP Apps to support resource reads and tool metadata access, beyond just tool invocation. This means MCP Apps running inside Codex can now pull data from MCP servers (reading resources) and inspect tool schemas (metadata), expanding the data access surface area. Endpoint monitoring should track what resources MCP Apps read and what tool metadata they access, as these represent new data flow paths from enterprise MCP servers into the Codex environment. Source: https://developers.openai.com/codex/mcp

- **[AIDR] [MEDIUM] OpenAI Codex Remote Workflow Egress WebSocket Transport**: Codex v0.120.0 introduces egress websocket transport for remote workflows with sandbox-aware filesystem APIs. This creates a new outbound data flow pattern — Codex agents can now establish persistent websocket connections from their sandbox to external endpoints, with filesystem access within the sandbox boundary. AIDR should monitor these outbound websocket connections as potential data exfiltration vectors. Source: https://developers.openai.com/codex/app-server
