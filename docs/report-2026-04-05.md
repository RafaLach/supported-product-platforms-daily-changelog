## Platform Details

### Google Vertex AI
- **RAG Engine Serverless Mode** (April 3, public preview): Fully managed database mode for RAG Engine that abstracts away database provisioning and scaling. Introduces new `RagManagedVertexVectorSearch` vector database type. Supports seamless transition between Serverless and Spanner deployment modes. Source: https://docs.cloud.google.com/vertex-ai/generative-ai/docs/rag-engine/serverless-mode
- **Veo 3.1 Lite** (April 2, public preview): Most cost-efficient Veo video generation model on Vertex AI. Completes the Veo 3.1 family (standard, Fast, Lite) — all with native audio generation. Source: https://cloud.google.com/blog/products/ai-machine-learning/veo-3-1-lite-and-a-new-veo-upscaling-capability-on-vertex-ai

### ChatGPT Enterprise (OpenAI)
- **Codex Plugin for Claude Code** (April 4): Official OpenAI MCP integration (`openai/codex-plugin-cc`) that allows Claude Code users to delegate code review, adversarial review, and background tasks to OpenAI Codex. Creates cross-provider data flow via Model Context Protocol. Source: https://github.com/openai/codex-plugin-cc
- **Codex Pay-as-You-Go Team Seats** (April 2): ChatGPT Business and Enterprise workspaces can now add Codex-only seats with usage-based pricing (no fixed seat fee). Introduces a new user seat type within enterprise workspaces. Source: https://openai.com/index/codex-flexible-pricing-for-teams/
- **ChatGPT in Apple CarPlay** (April 4, all plans): Voice-only ChatGPT access via Apple CarPlay on iOS 26.4+. Available on all plans including enterprise. New access surface for enterprise conversations. Source: https://help.openai.com/en/articles/20001153-using-chatgpt-on-carplay

### Cursor
- **Cursor 3 Interface** (April 2): Major release introducing three new capabilities:
  - **Agents Window**: Standalone workspace for running parallel agents across local, cloud, and remote SSH environments
  - **Design Mode**: Annotate and target UI elements directly in browsers for agent feedback
  - **Agent Tabs**: Side-by-side or grid view of multiple agent sessions
  - New `/worktree` command for isolated git worktrees, `/best-of-n` for parallel multi-model comparison, `Await` tool for monitoring long-running jobs, structured MCP content support
  - Source: https://cursor.com/changelog/3-0

### Dia Browser
- **v1.25.0** (April 2): Split-screen multitasking (drag tab to screen edge), custom tab icons (emoji favicons), Spotify integration visual (animated notes, hover for current track), download guidance animations. Source: https://diabrowser.com/release-notes
